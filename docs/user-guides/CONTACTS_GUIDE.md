# Contacts - User Guide

Manage your contacts efficiently with AiSHA CRM's user-friendly Contacts page.

## Searching and Filtering Contacts

1. **Search**: Enter keywords in the search bar to find specific contacts.
2. **Filters**:
   - **Status**: Filter contacts by their status.
   - **Assigned To**: Filter by the employee assigned to a contact.
   - **Tags**: Find contacts based on tags.

## Sorting Contacts

1. **Sort Options**:
   - Name A-Z
   - Name Z-A
   - Newest First
   - Oldest First
   - Last Updated

## Creating and Editing Contacts

1. **Create a New Contact**:
   - Click the "New Contact" button.
   - Fill out the form with details like name, email, phone, etc.
   - Click "Save" to create the contact.

2. **Edit an Existing Contact**:
   - Select the contact from the list.
   - Click on the contact's name or edit icon.
   - Make changes and click "Save".

## Linking Contacts

1. **Link a Contact to an Account**:
   - Edit a contact.
   - In the "Account" field, select or create an account.

2. **Assign a Contact to an Employee**:
   - Edit a contact.
   - In the "Assigned To" field, select an employee.

## Deleting Contacts

1. **Delete a Single Contact**:
   - Select the contact and click the delete icon.
   - Confirm deletion when prompted.

2. **Bulk Delete Contacts**:
   - Select multiple contacts using checkboxes.
   - Click "Clear Selection" if needed.
   - Click the trash can icon to delete selected contacts.

## Importing Contacts

1. **Import Contacts**:
   - Click the "Import" button.
   - Follow the prompts to upload a CSV file with contact details.

## No Matches Found

- If no contacts match your filters, you'll see "Try adjusting your filters".

By following these steps, you can effectively manage and utilize your contacts within AiSHA CRM.
